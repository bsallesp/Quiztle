# AWS Certified Cloud Practitioner (CLF-C02) - Exam Topics

## 1. Cloud Concepts (26%)
### 1.1 Definition of Cloud and its Value Proposition
- Understand the advantages of cloud computing (e.g., agility, elasticity, cost savings, and scalability).
- Recognize the different types of cloud computing models: public, private, and hybrid.

### 1.2 AWS Cloud Economics and Billing
- Explain the benefits of AWS’s pricing model (pay-as-you-go, reserved instances, and savings plans).
- Identify key cost management tools (AWS Cost Explorer, AWS Budgets, and AWS Cost and Usage Reports).

## 2. Security and Compliance (25%)
### 2.1 Shared Responsibility Model
- Understand the AWS Shared Responsibility Model: what AWS manages vs. what the customer manages (e.g., security *of* the cloud vs. security *in* the cloud).

### 2.2 AWS Cloud Security Concepts
- Overview of security services like AWS Identity and Access Management (IAM), AWS Organizations, AWS Shield, and AWS Key Management Service (KMS).
- Understand encryption at rest and in transit.

### 2.3 Compliance Programs
- Awareness of AWS compliance certifications and programs (e.g., PCI DSS, HIPAA, SOC, and GDPR).
- Understand how customers can leverage compliance information from AWS.

## 3. Technology (33%)
### 3.1 Core AWS Services
- **Compute**: Understand the function and use cases of EC2, Lambda, and Elastic Beanstalk.
- **Storage**: Understand S3, EBS, and Glacier, including their use cases.
- **Networking**: Understand VPC, Route 53, and CloudFront for managing networking in AWS.
- **Databases**: Understand RDS, DynamoDB, and their use cases.
- **Content Delivery and DNS**: Understand the purpose of services like CloudFront and Route 53.

### 3.2 AWS Global Infrastructure
- Understand AWS’s global infrastructure: Regions, Availability Zones, and Edge Locations.
- Understand the benefits of deploying resources across multiple regions.

### 3.3 AWS Well-Architected Framework
- Familiarity with the pillars of the AWS Well-Architected Framework: operational excellence, security, reliability, performance efficiency, and cost optimization.

## 4. Billing, Pricing, and Support (16%)
### 4.1 AWS Pricing Models
- Recognize various pricing models (e.g., On-Demand, Reserved Instances, and Spot Instances).
- Understand pricing considerations for core services like EC2, S3, and RDS.

### 4.2 Billing and Cost Management Tools
- Understand the use of AWS billing and cost management tools like AWS Cost Explorer, AWS Budgets, and consolidated billing.

### 4.3 AWS Support Plans
- Overview of the different support plans (Basic, Developer, Business, and Enterprise).
- Understand the features of each plan, including AWS Trusted Advisor and access to technical support.

### 4.4 AWS Marketplace
- Understand how to purchase third-party software and services through AWS Marketplace.
